# Sith Empire vs Unity Battle Simulator

This repository contains early experiments for a battle simulator between the Sith Empire and the Unity faction. 

- `battle_maps.js` lists eight battlefields with optional hazards and interactive objects.
- `sith_unity_chess.html` is a prototype chess mini-game using faction-themed pieces.

The map definitions are designed to support both small (5v5) and large (50+) unit battles with optional environmental hazards.
